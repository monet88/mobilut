# Domain: technology

## Purpose
Document technology decisions for graphics and rendering features, especially for LUT tooling in the mobile app.

## Scope
Included in this domain:
- Graphics rendering frameworks and shader usage
- Asset format choices for color transforms
- Media processing capabilities related to LUT application

Excluded from this domain:
- General product management decisions unrelated to rendering
- Backend infrastructure choices

## Ownership
Graphics Engineering Team

## Usage
Add new entries whenever long-lived rendering or asset decisions are locked for the LUT experience.
